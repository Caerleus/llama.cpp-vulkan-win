# Results Folder

This directory is intended to store the outputs and logs generated by llama.cpp.

## What to put here:
* **Benchmark Logs:** If you run performance tests using `llama-bench.exe`, save the results here to compare different models or GPU settings.
* **Server Logs:** You can redirect the output of the server script to a text file in this folder for debugging purposes.
* **Inference Outputs:** If you use scripts to generate long texts or data, this is the recommended place to save them.

## Why keep it?
Keeping your results organized helps you track which models perform best on your specific hardware configuration (Vulkan).
